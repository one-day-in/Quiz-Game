# КВИЗ-ШОУ — локальный сервер (с сохранением медиа в папку проекта)

## Что изменено
- Изображения/аудио/видео загружаются на локальный сервер и сохраняются в папку `media/`.
- В `localStorage` теперь хранятся только тексты/пути к файлам (а не base64).
- Добавлен скролл в модальных окнах и защита от "вылазящих" ячеек.

## Запуск
1) Установить зависимости:
```bash
npm install
```

2) Запустить сервер:
```bash
npm start
```

3) Открыть в браузере:
http://localhost:3000

## Где лежат файлы
Все загруженные медиа-файлы появляются в папке:
- `./media/`

> Примечание: тексты вопросов/ответов и ссылки на медиа по-прежнему сохраняются в `localStorage` браузера.
